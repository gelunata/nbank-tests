Перевод денег с одного аккаунта на другой
Депозит денег пользователем

ПРЕД. ШАГИ:
- Залогиниться Админом
- Создать пользователя
- Выйти из профиля Админа
- Залогиниться под созданным пользователем
- Создать аккаунт нажав на кнопку Create New Account
- Создать еще один аккаунт нажав на кнопку Create New Account
- Сделать депозит на один из аккаунтов через Deposit Money
- Начать кнопку Make a Transfer
- Сделать перевод
- Нажать на кнопку Transfer Again

Позитивные кейсы:
1 Пользователь может повторить перевод
ШАГИ:
- Нажать кнопку Repeat
- Корректно заполнить все поля
- Нажать на кнопку Send Transfer
ОЖИДАНИЕ:
- Сообщение об успешном переводе
- После нажатия Ok остаемся в окне Make a Transfer
- Перевод совершен через API

2 Фильтр переводов по имени работает корректно
ШАГИ:
- Нажать кнопку Repeat
- Ввести имя, с которым был совершен перевод
- Нажать кнопку Search Transactions
ОЖИДАНИЕ:
- В окне отобразятся переводы по заданному имени

3 Фильтр переводов по имени работает корректно, если нет совпадений
ШАГИ:
- Нажать кнопку Repeat
- ввести имя, с которым не было совершено перевода
- Нажать кнопку Search Transactions
ОЖИДАНИЕ:
- Появится сообщение, что нет совпадений
- После нажатия на кнопку Ok список переводов будет пустым

Негативные кейсы:
1 Пользователь не может повторить перевод
ШАГИ:
- Нажать кнопку Repeat
- Корректно заполнить все поля, но, например, сумму перевода сделать больше, чем есть на счету
- Нажать на кнопку Send Transfer
ОЖИДАНИЕ:
- Сообщение о том что перевод не совершен
- После нажатия Ok остаемся в прежнем окне Repeat Transfer
- перевод не совершен через API

2 Кнопка не активна, если не заполнены все поля в Repeat Transfer
ШАГИ:
- Заполнить все кроме одного поля
ОЖИДАНИЕ:
- Кнопка Send Transfer активна
