Изменение имени пользователя

ПРЕД. ШАГИ:
- Залогиниться Админом
- Создать пользователя
- Выйти из профиля Админа
- Залогиниться под созданным пользователем
- Нажать на имя профиля, тем самым перейдя в окно Edit Profile

Позитивный кейс:
1 Пользователь может изменить имя
ШАГИ:
- Вести корректное имя
- Нажать кнопку Save Changes
ОЖИДАНИЕ:
- Сообщение об успешном изменении имени
- После нажатия Ok имя профиля должно измениться (!!! Не изменяется, пока не обновишь страницу!!!)
- Остаемся в том же окне Change Profile
- Имя изменилось через API

Негативный кейс:
1 Пользователь не может изменить имя
ШАГИ:
- Вести не корректное имя
- Нажать кнопку Save Changes
ОЖИДАНИЕ:
- Сообщение об том, что невозможно поменять имя, так как не соответствует требованиям
- После нажатия Ok имя профиля не изменилось
- Остаемся в том же окне Change Profile
- Имя не изменилось через API
