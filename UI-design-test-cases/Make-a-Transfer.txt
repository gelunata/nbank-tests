Перевод денег с одного аккаунта на другой

ПРЕД. ШАГИ:
- Залогиниться Админом
- Создать пользователя
- Выйти из профиля Админа
- Залогиниться под созданным пользователем
- Создать аккаунт нажав на кнопку Create New Account
- Создать еще один аккаунт нажав на кнопку Create New Account
- Сделать депозит на один из аккаунтов через Deposit Money
- Начать кнопку Make a Transfer

Позитивные кейсы:
1 Пользователь может сделать перевод
ШАГИ:
- Корректно заполнить все поля
- Нажать на кнопку Send Transfer
ОЖИДАНИЕ:
- Сообщение об успешном переводе
- После нажатия Ok остаемся в окне Make a Transfer
- перевод совершен через API

Негативные кейсы:
1 Пользователь не может сделать перевод
ШАГИ:
- Корректно заполнить все поля, но, например, сумму перевода сделать больше, чем есть на счету
- Нажать на кнопку Send Transfer
ОЖИДАНИЕ:
- Сообщение о том что перевод не совершен
- После нажатия Ok остаемся в прежнем окне Send Transfer
- перевод не совершен через API

2 Пользователь не может сделать депозит если не заполнено одно из полей
ШАГИ:
- Заполнить все кроме одного поля
- Нажать на кнопку Send Transfer
ОЖИДАНИЕ:
- Сообщение о том что необходимо заполнить все поля
- После нажатия Ok остаемся в прежнем окне Send Transfer
