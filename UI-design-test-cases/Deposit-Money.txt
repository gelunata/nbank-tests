Депозит денег пользователем

ПРЕД. ШАГИ:
- Залогиниться Админом
- Создать пользователя
- Выйти из профиля Админа
- Залогиниться под созданным пользователем
- Создать аккаунт нажав на кнопку Create New Account
- Начать кнопку Deposit Money

Позитивные кейсы:
1 Пользователь может сделать депозит
ШАГИ:
- Выбрать аккаунт
- Ввести корректное сумму
- Нажать на кнопку Deposit
ОЖИДАНИЕ:
- Сообщение об успешном депозите
- После нажатия Ok переход на окна Главное окна
- депозит совершен через API

Негативные кейсы:
1 Пользователь не может сделать депозит
ШАГИ:
- Выбрать аккаунт
- Ввести не корректную сумму
- Нажать на кнопку Deposit
ОЖИДАНИЕ:
- Сообщение о том что депозит не совершен
- После нажатия Ok переход остаемся в окне Deposit Money
- депозит не совершен через API

2 Пользователь не может сделать депозит если не выбран аккаунт
ШАГИ:
- Ввести корректную сумму
- Нажать на кнопку Deposit
ОЖИДАНИЕ:
- Сообщение о том что депозит не совершен
- После нажатия Ok переход остаемся в окне Deposit Money
